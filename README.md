# Website Topup Game Online

Website topup game online yang bagus dan menarik.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Pilih Game
- Masukkan ID
- Pilih Nominal
- Bayar

## Technologies

- Next.js
- TypeScript
- Tailwind CSS
